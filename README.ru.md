## SMF Faker Generator

Этот пакет предназначен для заполнения таблиц в базе данных фейковыми данными SMF и Light Portal.

### Использование

* Распакуйте файлы пакета в любую директорию.
* Получите API на сайте https://www.pexels.com/api/key/ (для генерации случайных изображений).
* Настройте параметры подключения к базе данных в файле `.env` (см. пример в `.env.example`).
* Там же укажите `PEXELS_API_KEY`.
* Выберите, что именно генерировать, в файле `database/seeders/DatabaseSeeder.php` (закомментируйте лишнее).
* Выполните в консоли следующие команды:

    `composer install`

    `php artisan migrate:fresh --seed`

* Создайте снимки отдельных таблиц или всей базы данных:

    `php artisan snapshot:create --table=smf_boards`

    `php artisan snapshot:create`

* После этого можно импортировать нужные данные в вашу базу данных на реальном форуме.
