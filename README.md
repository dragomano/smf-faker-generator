## SMF Database Faker

Этот пакет предназначен для генерации таблиц с фейковым содержимым для тестирования SMF и Light Portal.

### Использование

Настройте параметры подключения к базе данных в файле `.env`.

Выберите, что именно генерировать, в файле `database/seeders/DatabaseSeeder.php`.

Выполните в консоли следующую команду:

`php artisan migrate:fresh --seed`
